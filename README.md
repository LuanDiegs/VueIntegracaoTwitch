# VueIntegracaoTwitch

Como configurar e inicializar o projeto.

## IDE recomendada

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur).

## Setup do projeto

```sh
npm install
```

### Rodar o programa

```sh
npm start
```
